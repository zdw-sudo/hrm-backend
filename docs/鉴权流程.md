# JWT 鉴权流程

## 登录流程

```mermaid
sequenceDiagram
    participant 前端
    participant LoginController
    participant LoginService
    participant Redis
    participant AuthManager
    participant StaffDetailsService
    participant JwtUtil

    前端->>LoginController: POST /login/{验证码}
    LoginController->>LoginService: login(staff, code)
    LoginService->>Redis: 校验验证码
    LoginService->>AuthManager: authenticate(工号, 密码)
    AuthManager->>StaffDetailsService: loadUserByUsername
    StaffDetailsService-->>AuthManager: StaffDetails + 权限
    AuthManager-->>LoginService: 认证成功
    LoginService->>JwtUtil: generateToken
    LoginService-->>前端: ResponseDTO(data, token)
    前端->>前端: localStorage 存 token
```

## 请求鉴权流程

```mermaid
sequenceDiagram
    participant 前端
    participant JwtFilter
    participant JwtUtil
    participant StaffDetailsService
    participant SecurityContext
    participant Controller

    前端->>JwtFilter: Authorization: Bearer {token}
    JwtFilter->>JwtUtil: extractUsername(token)
    JwtFilter->>StaffDetailsService: loadUserByUsername
    JwtFilter->>SecurityContext: 写入 Authentication
    JwtFilter->>Controller: 放行
    Controller->>Controller: @PreAuthorize 校验权限
```

## 401 vs 403

| 场景 | 处理器 | 业务码 |
|------|--------|--------|
| 未带 Token / Token 无效 | AuthenticationEntryPointHandler | 1200 |
| 已登录但权限不足 | AccessDeniedExceptionHandler | 1300 |
